## 这一页能让你做什么

跑通 OmicsClaw 的第一个 demo 分析（无需准备数据）。读完后你会拥有：

- 一个可用的 conda 环境
- 一份 spatial-preprocess demo 的输出报告
- 对 `omicsclaw.py run` CLI 的基本认知

## 安装：四步

### 1. 克隆仓库

```bash
git clone https://github.com/zhou-1314/OmicsClaw.git
cd OmicsClaw
```

### 2. 创建 conda 环境（mamba-first）

```bash
bash 0_setup_env.sh
```

脚本优先使用 `mamba`，无 `mamba` 时回退到 `conda`。它会从 `environment.yml` 创建名为 **OmicsClaw** 的 conda 环境，并依次完成 R 包、生信命令行工具和全部 Python 可选扩展的安装。预计耗时 10–30 分钟（取决于网络和镜像速度）。

### 3. 构建 vendor 工具（如适用）

```bash
conda activate OmicsClaw
bash 0_build_vendored_tools.sh
```

当前该脚本为存根（stub），尚未编译任何工具，执行后立即返回。待后续在 `tools/` 下添加上游工具时，重新运行此步骤以完成编译和链接。

环境激活会持续整个 shell 会话；下面的步骤无需再次激活。

### 4. 验证安装

```bash
python omicsclaw.py list
```

命令执行后应看到按领域分组的技能列表（空间转录组、单细胞、基因组等 7 个领域），确认环境已就绪。

## 第一次调用：跑 demo

`--demo` 模式使用内置的合成数据集（无需提供任何输入文件），适合快速验证安装是否正常。下面的命令运行空间转录组预处理技能：

```bash
python omicsclaw.py run spatial-preprocess --demo --output /tmp/quickstart_demo
```

终端输出示例：

```
Running Spatial Transcriptomics skill: spatial-preprocess (demo mode)
Output: /tmp/quickstart_demo

INFO: Demo file not found, generating synthetic data
INFO: Saved processed data: /tmp/quickstart_demo/processed.h5ad
Preprocessing complete: 180 cells, 3 clusters
```

输出目录下包含各步骤的 HTML 报告和图像文件。

## 它不是什么

- **不是 GUI 安装包**：本页假设你使用命令行；OmicsClaw-App 桌面客户端是独立的安装流程，参见 ecosystem/omicsclaw-app
- **不是云部署指南**：所有命令面向本地 Linux/macOS；远端执行请参考 engineering/remote-execution 页
- **不覆盖 R 依赖问题排查**：R 包安装常见问题请查阅 [docs/_legacy/R-DEPENDENCIES.md](https://github.com/zhou-1314/OmicsClaw/blob/main/docs/_legacy/R-DEPENDENCIES.md)
