## 本领域解决什么问题

基因组学域覆盖 bulk DNA-seq 完整链路：从原始 FASTQ 质控与比对，到 SNV/Indel/SV/CNV 变异检测、VCF 操作、功能注释与单倍型 phasing，以及 ATAC/ChIP peak calling 和 de novo 组装。与 bulkrna/singlecell 域的区别在于：本域分析 DNA 层面的遗传变异与染色质可及性，而非转录层面的表达量。

## 典型分析流程 ASCII 图

```text
FASTQ (DNA-seq)
   │
   ▼
genomics-qc           ← FASTQ QC (FastQC / fastp / MultiQC)
   │
   ▼
genomics-alignment    ← 比对 (BWA-MEM / Bowtie2 / Minimap2) → BAM
   │
   ├─→ genomics-variant-calling    ← SNV / Indel (GATK / DeepVariant)
   │       │
   │       ▼
   │   genomics-vcf-operations     ← VCF 过滤 / 统计 (bcftools)
   │       │
   │       ▼
   │   genomics-variant-annotation ← 功能注释 (VEP / SnpEff)
   │       │
   │       ▼
   │   genomics-phasing            ← 单倍型 phasing (WhatsHap / SHAPEIT5)
   │
   ├─→ genomics-sv-detection       ← 结构变异 (Manta / Delly / Sniffles)
   ├─→ genomics-cnv-calling        ← 拷贝数 (CNVkit / Control-FREEC)
   ├─→ genomics-epigenomics        ← ATAC / ChIP peak calling (MACS3)
   └─→ genomics-assembly           ← de novo 组装 (SPAdes / Flye)
```

## 何时不适用本领域

- **RNA 层面分析**：用 [bulkrna](/docs/domains/bulkrna) 或 [singlecell](/docs/domains/singlecell)
- **scDNA-seq（单细胞 DNA）**：不覆盖
- **群体遗传学（GWAS / 系统发育）**：不覆盖
- **元基因组（metagenomics）**：不覆盖
