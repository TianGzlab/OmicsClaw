## 本领域解决什么问题

蛋白组学域覆盖基于质谱的蛋白质组完整链路：从原始 MS 数据质控与格式导入，到肽段/蛋白鉴定、LFQ/TMT/DIA 定量、差异丰度分析、PTM 翻译后修饰解析、通路富集及结构蛋白组（XL-MS），与基于抗体的蛋白定量（Olink/SOMAscan）方法不同，本域专注质谱驱动的高通量蛋白鉴定与定量。

## 典型分析流程 ASCII 图

```text
原始 MS 数据 (.raw / .mzML)
   │
   ▼
proteomics-data-import       ← 格式转换 (MaxQuant / DIA-NN / Spectronaut)
   │
   ▼
proteomics-ms-qc             ← MS 质量控制 (PTXQC / rawTools)
   │
   ▼
proteomics-identification    ← 肽段 / 蛋白鉴定 (MaxQuant / MS-GF+ / Comet)
   │
   ▼
proteomics-quantification    ← LFQ / TMT / DIA 定量 (DIA-NN / Skyline)
   │
   ├─→ proteomics-de           ← 差异丰度 (MSstats / limma / proDA)
   ├─→ proteomics-ptm          ← 翻译后修饰 (MSstatsPTM / A-score)
   ├─→ proteomics-enrichment   ← 通路富集 (STRING / g:Profiler)
   └─→ proteomics-structural   ← 结构蛋白组 / XL-MS (XlinkX / pLink)
```

## 何时不适用本领域

- **基于抗体的蛋白定量（Olink / SOMAscan）**：不覆盖
- **结构生物学（X 射线晶体 / cryo-EM）**：不覆盖
- **代谢组（小分子）**：用 [代谢组学](/docs/domains/metabolomics)
- **空间蛋白组学**：不覆盖
