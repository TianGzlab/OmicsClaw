## 本领域解决什么问题

Bulk RNA-seq 域覆盖从原始 FASTQ 到生物学结论的完整链路：FASTQ 质控、比对统计、计数矩阵 QC、差异表达（DESeq2）、通路富集、可变剪接、WGCNA 共表达网络、细胞类型解卷积、PPI 网络、生存分析及 TrajBlend bulk-to-sc 轨迹插值，区别于单细胞域的无单细胞分辨率样本级统计分析。

## 典型分析流程 ASCII 图

```text
FASTQ
   │
   ▼
bulkrna-read-qc            ← FASTQ QC
   │
   ▼
bulkrna-read-alignment     ← STAR / HISAT2 → counts
   │
   ▼
bulkrna-qc                 ← 计数矩阵 QC
   │
   ├─→ bulkrna-geneid-mapping    ← 基因 ID 转换
   ├─→ bulkrna-batch-correction  ← 批次效应校正
   ├─→ bulkrna-de                ← 差异表达 (DESeq2)
   ├─→ bulkrna-splicing          ← 可变剪接
   ├─→ bulkrna-coexpression      ← WGCNA 共表达
   ├─→ bulkrna-deconvolution     ← bulk → 细胞类型比例
   ├─→ bulkrna-enrichment        ← 通路富集
   ├─→ bulkrna-ppi-network       ← PPI 网络
   ├─→ bulkrna-survival          ← 生存分析
   └─→ bulkrna-trajblend         ← bulk → sc 轨迹插值
```
