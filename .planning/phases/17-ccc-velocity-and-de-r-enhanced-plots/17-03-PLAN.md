<!-- CSV schemas consumed by velocity.R. Extracted from sc_velocity.py _velocity_cell_summary_df(). -->

From sc-velocity figure_data/velocity_cells.csv (produced by _velocity_cell_summary_df):
  - cell_id: str — cell barcode
  - umap_1: float — UMAP dimension 1 (present when X_umap exists in adata)
  - umap_2: float — UMAP dimension 2
  - velocity_magnitude: float — L2 norm of velocity vector (present when 'velocity' layer exists)
  - latent_time: float — latent time (present only for dynamical mode; may be absent)

IMPORTANT: velocity_cells.csv does NOT contain per-cell velocity direction vectors (dx, dy in UMAP space).
The layer-level velocity vectors are not exported to CSV because they are gene-space vectors, not UMAP-space.
Therefore velocity.R cannot draw per-cell arrows. Instead:

Primary mode ("grid"):
  Bin cells into a uniform grid over UMAP space (default 20x20 bins).
  Within each bin, compute mean velocity_magnitude and mean cell position.
  Draw a radial "burst" visualization: each bin shown as a dot sized by mean magnitude,
  colored by magnitude percentile via viridis. This is the ggplot2-native approximation of
  the velocity grid plot. It conveys spatial velocity magnitude distribution without requiring
  UMAP-space velocity direction.

Secondary mode ("scatter", used when umap_1/umap_2 absent):
  Fall back to a 1D strip plot of velocity_magnitude distribution across cells.

Optional coloring: if `color_by` param matches a column in the CSV (e.g. "latent_time"),
  use that column for the viridis color scale instead of velocity_magnitude.

From registry.R — renderer function signature:
  fn(data_dir, out_path, params)
  # params may include: plot_type (default "grid"), color_by, n_bins (default 20)

From common.R:
  theme_omics(), omics_palette(n, type="continuous"), ggsave_standard(plot, path, ...)

**Function: plot_velocity(data_dir, out_path, params)**

Libraries needed (load inside tryCatch): ggplot2 (already in common.R), dplyr. No additional packages required.

Step 1 — Load data:
```r
csv_path <- file.path(data_dir, "velocity_cells.csv")
if (!file.exists(csv_path)) stop("velocity_cells.csv not found in ", data_dir)
df <- read.csv(csv_path, stringsAsFactors = FALSE)
if (nrow(df) == 0) stop("velocity_cells.csv is empty")
```

Step 2 — Detect available columns:
```r
has_umap   <- all(c("umap_1", "umap_2") %in% colnames(df))
has_mag    <- "velocity_magnitude" %in% colnames(df)
has_lt     <- "latent_time" %in% colnames(df)
```

Step 3 — Determine color_by column:
- `color_by` param: if provided and present in df, use it. Else if has_mag, use "velocity_magnitude". Else use first numeric column.
- Coerce color_by column to numeric. Replace NA with 0 for magnitude; keep NA for latent_time (shown as grey).

Step 4 — Plot mode:
- `plot_type` param (default "grid" when has_umap, else "scatter")
- If has_umap is FALSE: produce a simple histogram/density plot of velocity_magnitude with subtitle "UMAP coordinates not available — showing magnitude distribution". Save and return.

**Grid mode (plot_type = "grid" or "magnitude"):**
- `n_bins` param (default 20, integer)
- Compute bin assignments: `df$bin_x <- cut(df$umap_1, breaks=n_bins, labels=FALSE)`, same for umap_2.
- Aggregate: `bin_df <- dplyr::group_by(df, bin_x, bin_y) |> dplyr::summarise(x=mean(umap_1), y=mean(umap_2), mag=mean(.data[[color_by]], na.rm=TRUE), n=n(), .groups='drop')`
- Remove bins with NA mag.
- Percentile-clip: clip mag at 1st and 99th percentile to avoid outlier dominance.
- Base layer: scatter of all cells at 10% alpha in grey "#CCCCCC", size=0.3
- Grid layer: `geom_point(aes(x=x, y=y, color=mag, size=n), data=bin_df, alpha=0.85)`
- Color: `scale_color_viridis_c(option="plasma", name=color_by)`
- Size: `scale_size_continuous(range=c(1,5), guide="none")`
- Labels: `labs(x="UMAP 1", y="UMAP 2", title="RNA Velocity — grid summary", subtitle=paste0(color_by, " per ", n_bins, "x", n_bins, " grid bins"))`
- Theme: theme_omics()
- Width=8, height=6

**Scatter mode (fallback when no UMAP):**
- `ggplot(df, aes(x=seq_len(nrow(df)), y=.data[[color_by]])) + geom_point(alpha=0.3, size=0.5) + geom_smooth(method="loess", se=FALSE, color="#E41A1C") + labs(x="Cell index", y=color_by, title="Velocity magnitude distribution") + theme_omics()`
- Width=8, height=4

Save via ggsave_standard. Wrap entire function in tryCatch; on error: cat to stderr, quit(status=1).

Note: Do NOT attempt to produce actual stream/arrow plots from this CSV — the data does not contain UMAP-space velocity direction vectors. The grid magnitude summary is the correct output. Label the plot subtitle with "grid summary" to be transparent about what is shown.
