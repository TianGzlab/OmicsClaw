**Helper: .load_ccc_data(data_dir)**
Internal function (prefixed with `.`) used by both renderers:
1. Try reading `sender_receiver_summary.csv` from data_dir. If found and non-empty return it.
2. Else try `top_interactions.csv`. If found, aggregate to sender_receiver by grouping on source+target, computing mean(score) and n() as n_interactions. Return that.
3. If neither exists, stop("No CCC data found. Expected sender_receiver_summary.csv or top_interactions.csv in data_dir")
4. Ensure columns are named `source`, `target`, `score`, `n_interactions`. Coerce score to numeric, fill NA with 0.

**Function 1: plot_ccc_heatmap(data_dir, out_path, params)**

Builds a heatmap/dot matrix using ggplot2 (not ComplexHeatmap — this data is small and ggplot2 gives cleaner control for the tile+dot combo).

Logic:
- Load data via `.load_ccc_data(data_dir)`
- Get `plot_type` from params (default "heatmap"; also accepts "dot")
- Build complete sender x receiver grid: expand.grid of all unique cell types for both axes. Left-join data onto grid; fill missing score with 0.
- Order cell types: by total outgoing score descending (consistent ordering between axes).
- Color scale: `scale_fill_gradient2(low="#377EB8", mid="white", high="#E41A1C", midpoint=0, name="Score")` for heatmap. For dot: `scale_color_gradient(low="#DEEBF7", high="#08306B", name="Score")` + `scale_size_continuous(range=c(1,8), name="n_interactions")`
- Heatmap variant: `geom_tile(aes(fill=score)) + geom_text(aes(label=ifelse(score>0, round(score,2), "")), size=2.5, color="grey20")`
- Dot variant: `geom_point(aes(color=score, size=n_interactions))`
- Both: `labs(x="Receiver", y="Sender", title="Cell-Cell Communication")` + `theme_omics()` + `theme(axis.text.x=element_text(angle=45, hjust=1))`
- Size: `width=max(5, n_types*0.7+1.5)`, `height=max(4, n_types*0.7+1.5)`, capped at 14x14 inches
- Save with ggsave_standard

**Function 2: plot_ccc_network(data_dir, out_path, params)**

Builds an arc network diagram in ggplot2 using geom_curve (no circlize dependency).

Logic:
- Load data via `.load_ccc_data(data_dir)`. Filter to score > 0 rows.
- `min_score` param (default 0, i.e. show all). Filter `score >= as.numeric(params[["min_score"]] %||% 0)`.
- `top_n` param (default 20): keep top N edges by score.
- Compute node positions: arrange cell types in a circle. For N cell types, angle_i = 2*pi*(i-1)/N. Node coords: x = cos(angle), y = sin(angle).
- Node sizes: proportional to total outgoing + incoming score. `scale_size_continuous(range=c(3,12))`.
- Node colors: `omics_palette(n_types)` categorical.
- Edge width: proportional to score. `scale_linewidth_continuous(range=c(0.3, 3))`.
- Edge alpha: 0.6.
- For arcs: use `geom_curve(aes(x=x_from, y=y_from, xend=x_to, yend=y_to, linewidth=score, alpha=score), curvature=0.25, arrow=arrow(length=unit(0.12,"inches"), type="closed"))`. Directed arcs (sender to receiver). Color arcs by sender cell type.
- Node layer on top: `geom_point(aes(x=x, y=y, size=total_score, color=cell_type), data=node_df)`.
- Labels: `geom_text(aes(x=x*1.15, y=y*1.15, label=cell_type), data=node_df, size=3, check_overlap=TRUE)`.
- Aesthetics: `coord_equal() + theme_void() + theme(legend.position="right")`.
- Size: 8x8 inches. Save with ggsave_standard.

Error handling: wrap both functions in tryCatch; quit(status=1) on error. Empty data (no non-zero edges) should produce a blank plot with a "No interactions detected" subtitle, not an error.
