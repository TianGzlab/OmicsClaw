**Function 1: plot_de_volcano(data_dir, out_path, params)**

Reads `de_top_markers.csv` from data_dir. Column detection (must handle both CSV schemas):
- Gene col: `names` if present, else `gene`, else stop with clear message
- FC col: `logfoldchanges` if present, else `log2fc` if present, else `scores` (with warning that log2FC not available)
- Pval col: `pvals_adj` if present, else `padj` if present, else stop

Plot logic:
- Add `neg_log10_pval` column = -log10(pval + 1e-300) to avoid -Inf
- Add `direction` column: "up" if fc > 0 & padj < 0.05, "down" if fc < 0 & padj < 0.05, "ns" otherwise. Use `padj_thresh` param (default 0.05) and `fc_thresh` param (default 0.25) from params.
- Color: up = "#E41A1C" (red), down = "#377EB8" (blue), ns = "#CCCCCC" (grey)
- Top N genes to label per direction: `n_label` param (default 5). Select top by abs(fc) * neg_log10_pval product. Use ggrepel::geom_text_repel for labels.
- One facet per `group` column value using `facet_wrap(~ group)`. If only one group, no facet.
- Horizontal dashed line at -log10(padj_thresh). Vertical dashed lines at +/-fc_thresh.
- Theme: theme_omics(), legend title "Direction"
- Output: ggsave_standard with width=8, height=6 (or width=4*ncol, height=4*nrow for multi-group, capped at 20x16 inches)

Libraries needed: ggplot2 (already in common.R), ggrepel

**Function 2: plot_de_heatmap(data_dir, out_path, params)**

Reads `de_top_markers.csv`. Same column detection as above.

Plot logic (mirrors markers.R ComplexHeatmap pattern):
- Select top N genes per group: `n_top` param (default 5). Sort by FC descending within each group.
- Build gene x group matrix: value = FC (or scores if FC not available). Use tidyr::pivot_wider.
- Replace NA with 0 in matrix (genes not tested in some groups).
- Row split by `feature_split` = the group column of origin for each gene. This creates per-group gene blocks.
- Color scale: blue-white-red diverging via `circlize::colorRamp2(c(-2, 0, 2), c("#377EB8","white","#E41A1C"))`. Clip at -2/+2 since values are log2FC.
- Column annotations: use `omics_palette(n_groups)` for cell type colors.
- `Heatmap(mat, name="log2FC", col=col_fun, cluster_rows=FALSE, cluster_columns=FALSE, row_split=row_split, column_title="Cell Type", row_title_gp=gpar(fontsize=8), show_row_names=TRUE, row_names_gp=gpar(fontsize=7), column_names_gp=gpar(fontsize=9))`
- Save: `png(out_path, width=max(6, ncol*0.7+2), height=max(4, nrow*0.18+2), units="in", res=200); draw(ht); dev.off()`

Libraries: ComplexHeatmap, circlize, tidyr, dplyr

Error handling: wrap both functions in tryCatch; on error write to stderr and quit(status=1). If CSV not found or empty, write clear error and quit(status=1).
