# 实验计划：组织驻留记忆CD8 T细胞多样性空间印记研究

## 项目概述

### 研究背景
本研究基于Nature论文《Tissue-resident memory CD8 T cell diversity is spatiotemporally imprinted》，旨在复现论文中的空间转录组分析结果，并挖掘新的生物学insight。论文揭示了小肠中TRM细胞在绒毛上部和下部的空间分布差异，以及这种多样性由肠道结构的区域化信号所驱动。

### 用户目标
1. **复现目标**：复现论文中的关键空间分析结果，包括细胞聚类、空间模式识别、结构轴定义等
2. **创新目标**：挖掘新的生物学insight，如细胞亚群、空间模式、基因表达特征等

### 数据信息
- **数据文件**：`/data1/TianLab/zhouwg/project/OmicsClaw/data/data/adata/visium_hd.h5ad`
- **数据规模**：91,716个细胞，13,735个基因
- **现有注释**：cells, n_genes_by_counts, log1p_n_genes_by_counts, total_counts, log1p_total_counts, pct_counts_in_top_50_genes, pct_counts_in_top_100_genes, pct_counts_in_top_200_genes, pct_counts_in_top_500_genes, n_genes

### 约束条件
- 仅使用提供的h5ad文件，不下载额外数据
- 使用OmicsClaw的现有分析技能
- 优先使用轻量级方法，深度学习作为可选高级阶段
- 确保结果可重现

## 分阶段实验设计

### 阶段1：数据质量评估与预处理（基线建立）
**目标**：评估数据质量，进行基本预处理，建立分析基线

#### 任务1.1：数据加载与初步检查
- **技能**：`spatial-preprocess`
- **参数**：
  - 输入文件：`visium_hd.h5ad`
  - 数据平台：Visium HD（自动检测）
  - **质量控制参数（基于论文方法调整）**：
    - 最小基因数：20（匹配论文的n < 20 total transcripts过滤）
    - 最大基因数：800（匹配论文的n > 800 total transcripts过滤）
    - 最小细胞数：3（每个基因至少在3个细胞中表达）
    - 线粒体基因百分比：<20%（如果数据包含mt基因）
  - 标准化：log1p标准化
  - 高变基因：2,000个（使用`seurat_v3`方法选择）
  - PCA：50个主成分
  - 聚类：Leiden聚类，resolution=0.8（初始探索），使用`igraph` flavor
- **成功信号**：
  - 成功加载91,716个细胞和13,735个基因
  - QC后保留70-90%的细胞（约64,000-82,000细胞）
  - 识别出5-15个初始聚类
  - 生成UMAP和空间聚类图
- **预期输出**：
  - `processed.h5ad`：预处理后的数据
  - QC报告：细胞/基因过滤统计、QC指标分布图
  - 可视化：UMAP图、空间聚类图、基因表达分布图

#### 任务1.2：数据质量深度评估
- **技能**：`spatial-statistics`
- **参数**：
  - 分析类型：`neighborhood_enrichment`（邻域富集分析）
  - 聚类键：`leiden`
  - 空间邻居数：6
- **成功信号**：
  - 检测到显著的空间聚类模式（z-score > 2）
  - 生成邻域富集热图
- **预期输出**：
  - 邻域富集分析结果
  - 空间聚类统计报告

### 阶段2：细胞类型注释与验证
**目标**：对细胞进行类型注释，验证与论文结果的一致性

#### 任务2.1：基于标记基因的细胞类型注释
- **技能**：`spatial-annotate`
- **参数**：
  - 方法：`marker_based`（标记基因评分）
  - 标记基因集：TRM细胞相关标记（CD8A, CD8B, ITGAE, CD69, CXCR6等）
  - 评分阈值：>0.5
- **成功信号**：
  - 识别出CD8 T细胞群体
  - 检测到TRM细胞亚群（效应型和记忆型）
- **预期输出**：
  - 注释后的h5ad文件
  - 细胞类型空间分布图
  - 标记基因表达热图

#### 任务2.2：细胞类型空间分布分析
- **技能**：`spatial-domains`
- **参数**：
  - 方法：`leiden`（空间加权）
  - 空间权重：0.3
  - 分辨率：1.0（匹配论文参数）
  - 子聚类分辨率：1.2（匹配论文参数）
- **成功信号**：
  - 识别出10-15个空间区域
  - 检测到绒毛上部和下部的空间分区
- **预期输出**：
  - 空间区域划分结果
  - 区域特异性标记基因

### 阶段3：空间模式与结构轴分析
**目标**：复现论文中的结构轴定义和空间模式分析

#### 任务3.1：空间可变基因检测
- **技能**：`spatial-genes`
- **参数**：
  - 方法：`morans`（Moran's I空间自相关）
  - 空间邻居数：6
  - 显著性阈值：FDR < 0.05
- **成功信号**：
  - 识别出>100个空间可变基因
  - 包括已知的TRM相关基因
- **预期输出**：
  - 空间可变基因列表
  - 基因空间表达模式图

#### 任务3.2：结构轴模拟分析与IMAP生成
**注意**：由于缺乏WGA染色和基底膜标记数据，我们使用简化方法模拟结构轴

- **技能**：`spatial-statistics` + 自定义Python脚本
- **参数**：
  - 分析类型：`ripley`（Ripley's K函数）
  - 聚类键：`cell_type`
- **简化结构轴定义方法**：
  1. **纵向轴替代**：使用细胞到组织质心的径向距离
  2. **隐窝-绒毛轴替代**：基于上皮细胞密度梯度
  3. **上皮轴替代**：计算每个细胞到最近5个上皮细胞的平均距离
- **IMAP生成简化方法**：
  1. 使用双指数变换：`f(x) = asinh(x/5)` 模拟论文中的biexponential变换
  2. 核密度估计：使用高斯核密度估计可视化细胞分布
  3. 手动门控：基于已知细胞类型位置定义区域边界
- **成功信号**：
  - 生成可解释的二维空间表示
  - 重现绒毛上部和下部的空间分区
  - TRM细胞在IMAP中显示预期分布模式
- **预期输出**：
  - 简化结构轴定义结果
  - IMAP-like可视化图
  - 空间梯度热图

### 阶段4：差异表达与功能分析
**目标**：分析不同空间区域的基因表达差异和功能富集

#### 任务4.1：空间区域差异表达分析
- **技能**：`spatial-de`
- **参数**：
  - 方法：`wilcoxon`（Wilcoxon秩和检验）
  - 分组：基于空间区域或细胞类型
  - 最小细胞数：每组>50个细胞
- **成功信号**：
  - 识别出区域特异性标记基因
  - 发现TRM细胞亚型差异表达基因
- **预期输出**：
  - 差异表达基因列表
  - 火山图、热图

#### 任务4.2：通路富集分析
- **技能**：`spatial-enrichment`
- **参数**：
  - 方法：`gsea`（基因集富集分析）
  - 基因集：GO、KEGG、Reactome
  - 富集阈值：FDR < 0.05
- **成功信号**：
  - 识别出TRM细胞分化相关通路
  - 发现空间位置相关的信号通路
- **预期输出**：
  - 通路富集结果
  - 富集网络图

### 阶段5：细胞通讯与微环境分析（高级阶段）
**目标**：分析细胞间通讯和微环境相互作用

#### 任务5.1：细胞-细胞通讯分析
- **技能**：`spatial-communication`
- **参数**：
  - 方法：`liana`（LIANA+）
  - 配体-受体数据库：内置数据库
  - 空间权重：启用
- **成功信号**：
  - 识别出TRM细胞与上皮/基质细胞的相互作用
  - 发现空间特异的通讯模式
- **预期输出**：
  - 细胞通讯网络
  - 配体-受体相互作用热图

#### 任务5.2：微环境特征分析
- **技能**：`spatial-statistics`
- **参数**：
  - 分析类型：`co_occurrence`（共现分析）
  - 聚类键：`cell_type`
  - 距离区间：0-50μm, 50-100μm, 100-200μm
- **成功信号**：
  - 量化细胞类型空间共现模式
  - 识别微环境组成特征
- **预期输出**：
  - 细胞共现分析结果
  - 微环境组成图

## 迭代与优化策略

### 迭代触发条件
1. **QC失败**：如果>50%细胞被过滤，调整QC阈值
2. **聚类不佳**：如果聚类数<5或>30，调整分辨率参数
3. **注释困难**：如果主要细胞类型无法识别，尝试多种注释方法
4. **计算瓶颈**：如果内存不足，使用子采样或分批处理

### 参数优化循环
```
初始参数 → 运行分析 → 评估结果 → 调整参数 → 重新运行
```

### 成功标准
1. **技术成功**：所有分析步骤顺利完成，无致命错误
2. **生物学合理**：结果符合已知生物学知识
3. **可重现性**：所有分析可完全重现
4. **新发现**：至少识别出1-2个新的生物学insight
